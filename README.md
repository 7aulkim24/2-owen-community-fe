# Owen Community Frontend

AWS AI School 2기 김은수 - 커뮤니티 서비스 프론트엔드 구현 과제입니다.

## 프로젝트 개요
커뮤니티 서비스의 프론트엔드 기능을 HTML, CSS, Vanilla JavaScript만을 사용하여 구현하였습니다.

## 주요 구현 기능

### 1. 회원가입 및 로그인
- **로그인 (`login.html`)**: 이메일 및 비밀번호 유효성 검사, 로그인 성공 시 게시글 목록 페이지로 이동.
- **회원가입 (`signup.html`)**: 
  - 프로필 이미지 업로드 및 미리보기.
  - 이메일, 비밀번호, 닉네임 실시간 유효성 검사.
  - 비밀번호 확인 일치 여부 체크.

### 2. 게시글 목록 조회 (`posts.html`)
- **레이아웃**: Flexbox를 활용한 카드 형태의 게시글 목록 배치.
- **디자인 가이드 반영**: 
  - 제목 26자 제한 및 말줄임 처리.
  - 좋아요, 댓글, 조회수 통계 포맷팅 (1k, 10k 등).
  - 카드 내 제목과 작성자 정보를 나누는 전폭 구분선 적용.
- **무한 스크롤**: 스크롤이 페이지 하단에 도달할 때마다 추가 게시글(더미 데이터) 로드 기능.

### 3. 게시글 상세 조회 (`post-detail.html`)
- **내용 구성**: 
  - 게시글 제목, 작성자(프로필 이미지, 닉네임), 작성 시간, 본문 내용 및 이미지 표시.
  - 게시글 수정 및 삭제 버튼 제공 (삭제 시 확인 모달 노출).
- **통계 및 인터랙션**: 
  - 좋아요, 조회수, 댓글 수 표시 및 숫자 포맷팅(1k 등) 적용.
  - 좋아요 버튼 클릭 시 토글 기능 및 실시간 수치 반영 시뮬레이션.
- **댓글 시스템**: 
  - 댓글 목록 조회 및 개별 댓글 수정/삭제 기능.
  - 댓글 작성 시 입력값 여부에 따른 등록 버튼 활성화 제어.
  - 댓글 삭제 시 확인 모달 제공.

### 4. 회원정보 수정 및 관리
- **회원정보 수정 (`edit-profile.html`)**: 
  - 현재 프로필 이미지 확인 및 변경 기능 (미리보기 지원).
  - 이메일 주소 확인 (수정 불가).
  - 닉네임 유효성 및 중복 검사 (최대 10자).
  - 회원 탈퇴 기능 및 확인 모달 구현.
- **비밀번호 수정 (`edit-password.html`)**:
  - 비밀번호 복합 규칙 유효성 검사.
  - 비밀번호 확인 일치 여부 실시간 체크.
  - 수정 완료 시 토스트 메시지 안내.

### 5. 공통 스타일 및 UI/UX
- **헤더 프로필 드롭다운**: 헤더의 프로필 아이콘 클릭 시 회원정보수정, 비밀번호수정, 로그아웃 메뉴 노출 및 각 페이지 이동 기능.
- **인터랙션 컴포넌트**: 
  - **토스트 메시지**: 수정 성공 등 알림을 위한 하단 토스트 구현.
  - **모달**: 회원 탈퇴, 게시글 삭제 등 중요 동작 확인을 위한 중앙 모달 구현.
- **헤더 통일**: 모든 페이지에 동일한 높이(80px)와 중앙 정렬된 제목, 프로필 아이콘 배치.
- **반응형 고려**: 다양한 화면 크기에서도 유지되는 레이아웃 구현.
- **사용자 경험**: 버튼 활성화/비활성화 상태 시각화, 부드러운 호버 효과 적용.

## 폴더 구조
- `css/`:
  - `common.css`: 공통 스타일 및 컴포넌트 정의
  - `pages/`: 페이지별 고유 스타일 (`login.css`, `signup.css`, `posts.css`, `post-detail.css`, `edit-profile.css`, `edit-password.css`)
- `js/`:
  - `api.js`: fetch API를 활용한 통신 모듈
  - `pages/`: 각 페이지별 비즈니스 로직 (`login.js`, `signup.js`, `posts.js`, `post-detail.js`, `edit-profile.js`, `edit-password.js`)
  - `utils/`: 유효성 검사 등 공통 유틸리티 함수 (`validation.js`)
- `*.html`: 각 페이지별 마크업 파일 (`login.html`, `signup.html`, `posts.html`, `post-detail.html`, `edit-profile.html`, `edit-password.html`)
- `build-css.sh`: CSS 빌드 스크립트
- `package.json`: 프로젝트 설정 및 의존성 관리
