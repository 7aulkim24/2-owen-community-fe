# Owen Community Frontend

AWS AI School 2기 김은수 - 커뮤니티 서비스 프론트엔드 구현 과제입니다.

## 프로젝트 개요
커뮤니티 서비스의 프론트엔드 기능을 HTML, CSS, Vanilla JavaScript만을 사용하여 구현하였습니다.

## 주요 구현 기능

### 1. 회원가입 및 로그인
- **로그인 (`login.html`)**: 이메일 및 비밀번호 유효성 검사, 로그인 성공 시 게시글 목록 페이지로 이동.
- **회원가입 (`signup.html`)**: 
  - 프로필 이미지 업로드 및 미리보기.
  - 이메일, 비밀번호, 닉네임 실시간 유효성 검사.
  - 비밀번호 확인 일치 여부 체크.

### 2. 게시글 목록 조회 (`posts.html`)
- **레이아웃**: Flexbox를 활용한 카드 형태의 게시글 목록 배치.
- **디자인 가이드 반영**: 
  - 제목 26자 제한 및 말줄임 처리.
  - 좋아요, 댓글, 조회수 통계 포맷팅 (1k, 10k 등).
  - 카드 내 제목과 작성자 정보를 나누는 전폭 구분선 적용.
- **무한 스크롤**: 스크롤이 페이지 하단에 도달할 때마다 추가 게시글(더미 데이터) 로드 기능.

### 3. 공통 스타일 및 UI/UX
- **헤더 통일**: 모든 페이지에 동일한 높이(80px)와 중앙 정렬된 제목, 프로필 아이콘 배치.
- **반응형 고려**: 다양한 화면 크기에서도 유지되는 레이아웃 구현.
- **사용자 경험**: 버튼 활성화/비활성화 상태 시각화, 부드러운 호버 효과 적용.

## 폴더 구조
- `css/`: 전역 및 컴포넌트 스타일 (`style.css`)
- `js/`: 
  - `pages/`: 각 페이지별 비즈니스 로직 (`posts.js`, `login.js`, `signup.js`)
  - `utils/`: 유효성 검사 등 공통 유틸리티 함수
  - `api.js`: fetch API를 활용한 통신 모듈
- `assets/`: 이미지 및 아이콘 자원
